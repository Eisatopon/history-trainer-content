## 🧠 Δομή JSON

Κάθε αρχείο περιλαμβάνει:

- `book` → τίτλος βιβλίου  
- `enothta` → ενότητα  
- `questions` → λίστα ερωτήσεων  

Κάθε ερώτηση περιέχει:

- `id`
- `type` (multiple_choice / true_false)
- `difficulty`
- `question`
- `options` (αν υπάρχει)
- `correct`
- `explanation`
